# Predicting_Donor_Response
Using Machine Learning techniques to predict if a given person inside the Civil Support Alliance (CSA) database will donate, if contacted.


Data Mining II – Group Project (2025/2026)
Nova Information Management School – Universidade Nova de Lisboa

## Project Overview

Nonprofit organizations often struggle with decreasing engagement in fundraising campaigns due to donor fatigue caused by repeated and generic solicitations.

The Civic Support Alliance (CSA) aims to modernize its outreach strategy by using data-driven methods to identify individuals who are most likely to respond positively to donation requests.

This project develops machine learning models to predict whether a person will donate if contacted, enabling organizations to target the right individuals while reducing unnecessary outreach.

The task is framed as a binary classification problem using historical campaign data.
